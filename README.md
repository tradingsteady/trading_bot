# Trading Bot

This repository contains a Jupyter notebook and configuration file for a **trading bot**.

## Features

- Implementation of the trading bot logic  
- Configurable parameters via `config.py`  
- Direct communication between the bot and the broker API
- Calculating indicators and placing orders

## Included Files

- **bot.ipynb** – Main notebook implementing the trading bot  
- **config.py** – Configuration file for bot parameters and settings  

## Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/tradingsteady/trading_bot.git
    ```

2. **Open the Jupyter notebook:**

    ```bash
    jupyter notebook
    ```
